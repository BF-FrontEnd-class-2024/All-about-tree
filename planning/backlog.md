# Backlog

A collection of user stories you can choose from. Stories inside each priority
level are not necessarily in order, it's up to your group to decide how they fit
into your strategy. These are also just suggestion, feel free to change them or
create your own!

## Must-Haves

> these are necessary for basic usability

- [ ] As a user I want to `title` of the page

- there is a clear title of the page.

- [ ] As a user I want to `logo` of the page
- there is a clear logo of the page.

- [ ] As a user I want to `Description` of the page
- there is a description of page.

- [ ] As a user I want to `Navigation` of the page

- there different are navigation link of the page.

- [ ] As a user I want to `Social media links` of the page

- there are different social media links of the page.

## Should Haves

- [ ] As a user I want to `NavBar` of the page

- there is NavBar to Navigate deferent page.
- [ ] As a user I want to `About page` of the page

- there is About page to know more about us.
- [ ] As a user I want to `Contact us` of the page

- there is Contact us page to contact us.
